# 💻 Embedded C Programs

This repository contains basic to intermediate Embedded C programs for microcontroller programming using Arduino and STM32.

---

## 🔧 Topics Covered

- GPIO Programming (LED Blinking)
- Switch Interfacing
- Delay Functions
- PWM (Pulse Width Modulation)
- ADC (Analog to Digital Conversion)
- Interrupt Basics

---

## 🛠 Programs

### 🔹 LED Blinking
- Basic program to blink LED using GPIO  
- Demonstrates digital output control  

---

### 🔹 Switch Interfacing
- Reads input from push button  
- Controls LED based on switch state  

---

### 🔹 PWM LED Control
- Controls brightness of LED  
- Uses duty cycle variation  

---

### 🔹 ADC Example
- Reads analog input (e.g., sensor value)  
- Converts to digital output  

---

### 🔹 Interrupt Example
- Demonstrates interrupt handling  
- Executes function on external trigger
